Reproduction of a sokoban game.

my_sokoban 2021.

100% mouli.

USAGE
  ./my_sokoban map
  ./my_sokoban -h

OPTION
  map : display a map and the player can move in it.
  -h : help

DESCRIPTION
  map file representing the warehouse map, containing '#' for walls, 'P' for the player, 'X' for boxes and 'O' for storage locations.
